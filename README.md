# Data-Visualization-Dashboard-for-Global-Income-Inequality-Analysis
# 🌍 Global Income Inequality Analysis Platform

An advanced data analytics and visualization platform designed to explore, analyze, and visualize global income inequality trends using interactive dashboards and modern analytics tools.

This project combines the power of **Power BI** and **Streamlit** to transform complex economic datasets into meaningful insights through dynamic visualizations, comparative analytics, and AI-powered reporting features.

---

# 🚀 Project Overview

The Global Income Inequality Analysis Platform helps users understand:

- Global income distribution patterns
- Country-wise inequality comparisons
- Historical economic trends
- Regional development indicators
- Economic disparity analysis
- Data-driven socio-economic insights

The platform is developed with an interactive multi-page architecture for enhanced analytics and user experience.

---

# 📊 Power BI Dashboard

The Power BI dashboard provides:

- Interactive KPI dashboards
- Dynamic visual analytics
- Country and region-level filtering
- Trend analysis visualizations
- Comparative economic dashboards
- Business intelligence reporting
- Clean and responsive dashboard design

---

# 🌐 Streamlit Analytics Application

The Streamlit application enables users to interactively explore global inequality data through multiple analytical modules.

## 📄 Application Modules

| Module | Description |
|---|---|
| 🏠 Home | Platform overview and summary insights |
| 📊 Dashboard | Interactive visualization dashboard |
| 🌎 Country Compare | Country-wise economic comparison analysis |
| 📈 Trends | Historical trend and growth analysis |
| ⚙️ Settings | Application configuration settings |
| ℹ️ About | Project information and documentation |
| 📧 Email Reports | Automated report generation feature |
| 🤖 AI Insights | AI-powered analytical insights |
| 📜 Privacy | Privacy policy and data usage information |
